O que é Git?

O Git é um sistema de controle de versão distribuída de código aberto e gratuito projetado para lidar com tudo, desde projetos pequenos a grandes com velocidade e eficiência.

Git é fácil de aprender e tem uma pequena pegada com desempenho rápido relâmpago. Ele supera ferramentas SCM como Subversão, CVS, Perforce e ClearCase com recursos como ramificações locais baratas, áreas de estadiamento convenientese vários fluxos de trabalho.

Qual é a finalidade de GitHub Desktop?

Use o GitHub Desktop para gerenciar seus projetos, crie compromissos significativos e acompanhe o histórico do projeto em um aplicativo em vez de na linha de comando.
